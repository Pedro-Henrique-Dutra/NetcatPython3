# Netcat Python3

Implementação educacional inspirada no Netcat utilizando Python 3.

## Objetivos

Este projeto foi desenvolvido para aprofundar conhecimentos em:

- Redes TCP/IP
- Programação com sockets
- Linux
- Segurança da Informação
- Pentesting

## Funcionalidades

- Comunicação TCP cliente-servidor
- Múltiplas conexões
- Transferência de arquivos
- Execução remota de comandos
- Shell interativo

## Tecnologias

- Python 3
- Sockets
- Threading
- Linux

## Estrutura

```text
src/
docs/
tests/
```

## Aviso

Projeto desenvolvido exclusivamente para fins educacionais e laboratoriais.
